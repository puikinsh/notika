# Legacy Files Archive

This folder contains the **original Bootstrap 3 + jQuery versions** of all Notika template pages.

## Purpose

These files are preserved for:
- **Reference**: Compare old vs new implementations
- **Legacy support**: If anyone needs the original Bootstrap 3 versions
- **Historical record**: Track what was changed during modernization

## Status

All files in this folder use:
- ❌ Bootstrap 3.x (outdated)
- ❌ jQuery (no longer needed)
- ❌ Legacy CSS/JS loading
- ❌ Old HTML structure

## Modern Versions

The modern, Vite-enabled versions are in the parent directory:
- ✅ Bootstrap 5.3.8
- ✅ Vite 7.x bundling
- ✅ ES6 modules (no jQuery)
- ✅ Modern JavaScript

## Do Not Use

**These files are for reference only.** Do not use them in production or development.

Always use the modernized versions in the parent directory.
